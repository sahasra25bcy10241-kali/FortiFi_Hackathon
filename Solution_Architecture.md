# Solution Architecture

System Modules:

1. User Interface Layer
Receives URL input from user.

2. Validation Layer
Checks if URL format is valid.

3. Analysis Engine
Evaluates URL against predefined rules.

4. Risk Scoring Module
Assigns score based on suspicious indicators.

5. Decision Engine
Classifies URL as Safe or Suspicious.

6. Output Module
Displays result and explanation.


Data Flow:

User → Input URL → Validation → Analysis → Scoring → Classification → Result Display
